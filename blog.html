Website cherry = popped

January 6, 2019

My god has it been a long and tumultuous journey to get here… Not really I’m being dramatic. But anyways, this past semester (Fall 2018), I had made it a goal for myself to build my website and launch a blog from that website.

This was a lot harder than I imagined. I thought I would just type up some HTML using a premade format and I’d be done with it. Nope. I practically tore my hair out trying to get some of the javascript to work, since I know literally nothing about javascript (or jQuery?). In the end, I did learn a lot and I'm proud of how it all turned out.

I jumped back and forth between using Bootstrap4, a well-established front-end framework, and Materialize, a not so well-established, but very cool, front-end framework. I did a little bit of coding in both, but ended up deciding to go with Materialize. I did so for a few reasons.

  1. It’s based off Google’s Material Design Philosophy
  2. I really like Material Design
  3. It stands out from the crowd a little bit

I may have done more work than necessary, but that’s rather typical of me. It’s definitely been a learning experience. Oh, and this is also my first blog post. This is also my first time using Markdown to write anything (thanks Dillinger for making it so easy), I guess you could say there’s a lot of firsts going on here. New year, new me! I’m not quite sure what I’m gonna write about in the future, probably some random thoughts that pop in my mind. This might (will) range from nerdy stuff, politics, gaming, rants, LGBTQ+ issues, etc. I hope you stick around!
Bye. <3
